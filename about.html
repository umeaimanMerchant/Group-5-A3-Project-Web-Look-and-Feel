About BookBridge

Our Story

BookBridge was created by a team of passionate developers and book lovers who wanted to make literature more accessible to everyone. Our team consists of four members, each bringing their unique skills and perspectives to the project.

Our Mission

Making Books Accessible

Our mission is to make books accessible to everyone, regardless of their financial situation. We believe that everyone should have the opportunity to read and learn from books.

Building Community

We're not just about sharing books; we're about building a community of readers who can connect, discuss, and learn from each other.

Promoting Literacy

By making books more accessible, we're helping to promote literacy and education in our communities.

Logical Data Model

Data Relationships

The BookBridge database is built on a well-structured relational model to support efficient tracking and interaction:

  • Users → Books: A single user can list multiple books for sharing.
  • Books → Borrow Records: Each book can be borrowed multiple times, with every instance recorded.
  • Users → Borrow Records: Each borrow record links both the borrower and the owner.
  • Books → Reviews: Multiple users can review the same book, providing feedback and ratings.
  • Users → Reviews: Users can write reviews for different books they have interacted with.
  • Books → Book Images: Each book can have multiple images, with one marked as primary.
  • Users → Preferences: Each user has a single preferences record storing favorite genres and notification settings.

This relational setup enables BookBridge to handle user activity, book details, borrowing history, visual content, and personalized experiences effectively.
